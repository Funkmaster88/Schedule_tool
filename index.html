Google Maps API Key Required

Route Planner needs your Google Maps API key to function. This key is stored only on your computer — never stored in code or sent to servers.

How to get your API key:
1. Go to Google Cloud Console
2. Create a new project
3. Enable: Maps JavaScript API, Places API, Geocoding API, Distance Matrix API
4. Create an API key (Credentials → API Key)
5. Set HTTP referrer restrictions to your domain
Your key is stored locally in your browser — never shared or logged

Changelog

v18 — Full-day drive-aware assignment (current)
Replaced geographic clustering with a smarter per-job greedy algorithm. Assignment now uses full-day workload (AM + PM combined, including estimated drive time at 45 mph) as the primary cost function — eliminating the problem where one window was overloaded while another was empty. Gas+Sewer jobs are assigned to the best dual-certified inspector available, with a balance-triggered split pass as a fallback. Jobs are sorted heaviest-first so constrained jobs get first pick of inspectors. Balance pass now simulates actual workload change before committing any split, using full-day totals instead of per-window max. Fixed inspector scoring to blend workload + drive-to-job cost so nearby inspectors are preferred when workloads are close. Fixed export and assignment panel time labels to include drive time, not just service minutes.
v17 — Enhanced security & API rate limiting
Added HTTPS warning banner to alert users if served over insecure HTTP connection. Implemented HTML escaping (XSS prevention) for all user input (addresses, names, notes) before display. Added rate limiting on Distance Matrix API calls — minimum 60-second cooldown between drive time refinements to prevent accidental API overuse. Updated USER_GUIDE with comprehensive security section and Google Maps API cost breakdown. All security features work transparently without user action.
v16 — Secure API key input
Removed hardcoded Google Maps API key from code for security. App now prompts users to enter their own API key on first load, stored securely in browser local storage. No API keys exposed in GitHub repository. Each user brings their own key, eliminating security risk. Clear setup modal with instructions on obtaining API key from Google Cloud Console.
v15 — Smart splits & real drive times
Added transparency to split logic — now shows WHY each Gas+Sewer job was split (no dual-certified available, workload balance, etc.). Implemented post-assignment drive time refinement using Google Distance Matrix API for more accurate road-based times instead of straight-line estimates. Clarified that Radon and Mold are ALWAYS bonded together and never split. Improved Office/Midday Waypoint logic documentation — it chooses the closer location (home vs office) between AM and PM jobs for optimal routing.
v14 — Google Maps integration
Replaced Leaflet with Google Maps for better map quality and features. Added Google Places Autocomplete to address inputs — eliminates typos and ensures accurate address matching. Upgraded geocoding to use Google Geocoding API for more reliable address-to-coordinates conversion. Integrated Distance Matrix API for calculating actual road-based driving times. Improved markers with custom canvas-drawn icons for better visual clarity on the map.
v13 — Property types, split reassignment & bulk import
Added property type selector (Single Family, Duplex, Multifamily) when creating jobs. Duplex and Multifamily now add 30 minutes to gas inspection time for more accurate estimates. Split gas+sewer jobs now show individual reassignment controls — reassign just the sewer portion to a different inspector without affecting the gas assignment. Bulk import feature lets you paste Salesforce schedules directly — automatically extracts Team Radon, Team Sewer, and Team Gas jobs, combines duplicate addresses into single jobs with multiple services, and determines AM/PM windows based on job time slots.
v12 — Balanced workload optimization
Fixed PM window assignment to consider full-day balance (AM + office + PM). Smarter midday break logic — chooses home or office based on distance from last AM job. Improved gas+sewer split to prefer splitting when requested jobs or light sewer inspectors would benefit. Better geocoding for St. Louis metro area. Added changelog modal window for cleaner export tab.
v11 — Time-balanced routing
Fixed major bug where one inspector could receive 5+ hrs of work in a 3-hr window. Algorithm now balances by estimated minutes (service time + drive time) instead of job count. Heaviest jobs assigned first. Summary cards show time range instead of job counts.
v10 — Address geocoding fallback + inspector exclusions
Added multi-attempt geocoding for newer addresses (tries 5 query formats before giving up). Added excluded inspector dropdown per job — prevents specific inspectors from being assigned regardless of availability.
v9 — Stability fix
Rewired all event listeners to fire after page load. Fixed map not rendering and inputs not responding in embedded iframe context.
v8 — Five feature upgrade
Added unavailable toggle (⊘) per inspector for vacation/sick days. Added notes field on jobs. Added manual reassign dropdown in Assignments tab. Added Export tab with clipboard copy. Added per-inspector job cap (later removed).
v7 — Inline editing
Added edit (✎) button on every inspector and job. Inline edit forms pre-fill all current values. Address changes trigger re-geocoding automatically. Editing a job clears its assignment to prevent stale routing data.
v6 — AM/PM windows + cert splitting
Added AM (9–12) and PM (2–5) window toggle per job. Assignments view grouped by window. When a job needs both Gas and Sewer, algorithm attempts to split across two inspectors rather than pile both on one dual-certified inspector.
v5 — Radon and Mold certifications
Added Radon and Mold cert toggles alongside Gas and Sewer on both inspectors and jobs. All four feed into cert-matching logic. Color coded: Gas=amber, Sewer=green, Radon=purple, Mold=coral.
v4 — Tile provider fix
Switched map tiles from OpenStreetMap to CartoDB to resolve 403 referer-blocked errors in embedded environments.
v3 — Load balancing
Replaced pure distance-based assignment with load-balanced greedy: fewest jobs first, distance as tiebreaker. Added assignment summary cards (total jobs, active inspectors, spread, balance verdict). Added load bar on each inspector chip.
v2 — Certifications + inspector request/exclude
Added Gas and Sewer cert toggles per inspector and per job. Only certified inspectors are eligible for cert-required jobs. Added requested inspector dropdown (overrides auto-assign). Cert conflict flagging when no certified inspector is available.
v1 — Initial build
Basic dispatch tool with St. Louis map (Leaflet.js). Inspector home address roster with geocoding. Job address input with auto-assignment to closest inspector by straight-line distance. Color-coded map pins and dashed routing lines.
