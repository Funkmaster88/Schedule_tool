Changelog

v11 — Time-balanced routing (current)
Fixed major bug where one inspector could receive 5+ hrs of work in a 3-hr window. Algorithm now balances by estimated minutes (service time + drive time) instead of job count. Heaviest jobs assigned first. Summary cards show time range instead of job counts.
v10 — Address geocoding fallback + inspector exclusions
Added multi-attempt geocoding for newer addresses (tries 5 query formats before giving up). Added excluded inspector dropdown per job — prevents specific inspectors from being assigned regardless of availability.
v9 — Stability fix
Rewired all event listeners to fire after page load. Fixed map not rendering and inputs not responding in embedded iframe context.
v8 — Five feature upgrade
Added unavailable toggle (⊘) per inspector for vacation/sick days. Added notes field on jobs. Added manual reassign dropdown in Assignments tab. Added Export tab with clipboard copy. Added per-inspector job cap (later removed).
v7 — Inline editing
Added edit (✎) button on every inspector and job. Inline edit forms pre-fill all current values. Address changes trigger re-geocoding automatically. Editing a job clears its assignment to prevent stale routing data.
v6 — AM/PM windows + cert splitting
Added AM (9–12) and PM (2–5) window toggle per job. Assignments view grouped by window. When a job needs both Gas and Sewer, algorithm attempts to split across two inspectors rather than pile both on one dual-certified inspector.
v5 — Radon and Mold certifications
Added Radon and Mold cert toggles alongside Gas and Sewer on both inspectors and jobs. All four feed into cert-matching logic. Color coded: Gas=amber, Sewer=green, Radon=purple, Mold=coral.
v4 — Tile provider fix
Switched map tiles from OpenStreetMap to CartoDB to resolve 403 referer-blocked errors in embedded environments.
v3 — Load balancing
Replaced pure distance-based assignment with load-balanced greedy: fewest jobs first, distance as tiebreaker. Added assignment summary cards (total jobs, active inspectors, spread, balance verdict). Added load bar on each inspector chip.
v2 — Certifications + inspector request/exclude
Added Gas and Sewer cert toggles per inspector and per job. Only certified inspectors are eligible for cert-required jobs. Added requested inspector dropdown (overrides auto-assign). Cert conflict flagging when no certified inspector is available.
v1 — Initial build
Basic dispatch tool with St. Louis map (Leaflet.js). Inspector home address roster with geocoding. Job address input with auto-assignment to closest inspector by straight-line distance. Color-coded map pins and dashed routing lines.
